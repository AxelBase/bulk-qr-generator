QR Code Versions, Module Size & Minimum Print Requirements

Published: November 22, 2025

You’ve seen QR codes that “don’t scan” on business cards or posters. 99% of the time, it’s because they were printed too small.

QR Code Versions (1–40)

Version 1 = 21×21 modules
Version 40 = 177×177 modules
Longer URLs → higher version → more modules → needs larger physical size.

The Golden Rule: Minimum Module Size

Each black/white square (module) must be at least 0.5mm × 0.5mm (2px at 300 DPI) for reliable scanning.

Real Minimum Print Sizes

URL LengthTypical VersionMin Print Size (mm)Min Print Size (inches)
Short (~50 chars)V10–1520 × 20 mm0.8" × 0.8"
Medium (~150 chars)V20–2525 × 25 mm1.0" × 1.0"
Long (~300+ chars)V30–4035 × 35 mm1.4" × 1.4"

Quiet Zone (White Border)

Always include 4 modules of white space around the QR. No text, logos, or colors in this zone.

Pro Tips for Print

  • Use SVG for print — infinitely sharp
  • Aim for 30–40mm for most real-world uses
  • Test with multiple phones before mass printing
  • Avoid glossy surfaces that cause glare

A QR code is only useful if it scans — size and contrast are everything.
