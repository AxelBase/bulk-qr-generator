About Bulk QR Code Generator

Bulk QR Code Generator is a 100% client-side, open-source web utility that lets you create thousands of fully customizable QR codes from a single CSV file — instantly, privately, and offline.

Built with SvelteKit and deployed via GitHub Pages, this tool uses only modern browser APIs: kjua for QR rendering, PapaParse for lightning-fast CSV parsing, and JSZip for on-the-fly ZIP creation. No backend. No database. No tracking.

Whether you're organizing a 5,000-person conference, tagging warehouse inventory, distributing personalized marketing links, or printing WiFi access codes for a café — this tool eliminates the pain of manual QR creation. Your CSV stays on your device. Generated QR codes are packaged into clean ZIP files (PNG or SVG) and downloaded directly — all in under 30 seconds.

Privacy by design: Because everything runs locally, we never see your data. No IP logging, no analytics, no cookies. After the first load, it works completely offline. This makes it ideal for sensitive environments: corporate intranets, secure facilities, or when internet access is unreliable.

Performance is unmatched. Thanks to cooperative multitasking, chunked async processing, and memory-efficient blob handling, the tool can generate 10,000+ QR codes without freezing your browser — even on modest laptops. We've tested up to 20,000 rows successfully.

Full customization gives you professional results: use any CSV column as filename, add automatic URL prefixes/suffixes, choose from four error-correction levels (L–H), set exact sizes (200–500px), and pick any foreground/background color. Output as crisp SVG for print or lightweight PNG for digital use.

Deploy your own branded version in 60 seconds using GitHub Pages. Fork the repo, change colors and logo, and you're live. No build tools needed.

This isn't just a QR generator — it's a privacy-first productivity powerhouse trusted by event organizers, warehouse managers, educators, and developers worldwide.

How to Use Bulk QR Code Generator

1. Prepare Your CSV

Create a simple CSV file with clear headers. At minimum, include one column containing the data you want inside each QR code (e.g., URLs, text, vCard info). Optionally add:

  • Filename column — e.g., "name", "ticket_id"
  • Prefix/suffix columns — or set globally

Example:
name,url
John Doe,https://event.com/123

2. Configure Settings

Drag & drop your CSV or paste directly. Then customize:

  • Select content column (what goes inside QR)
  • Choose filename column (optional)
  • Add prefix/suffix (e.g., https://mystore.com/)
  • Set size, error correction (H for logos), and colors

Preview first 5 codes to verify everything looks perfect.

3. Generate & Download

Click “Generate All QR Codes”. Watch the live progress bar. When complete:

  • Download PNG ZIP — perfect for email, web, WhatsApp
  • Download SVG ZIP — infinitely scalable for print, stickers, signage

Pro tip: Use SVG for anything printed. Use PNG for digital sharing.
